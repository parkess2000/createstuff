# This file explains how to run the example code in the CreateStuff repository.
# The objective of the code is to demonstrate how to use an existing publicly
# available dataset. 

Step 1: Open file "iris_dataset_example.py"
Step 2: Copy code in file
Step 3: Open Browser with the following URL
        https://www.katacoda.com/courses/tensorflow/playground
Step 4: Cut and paste code into katacoda code editor
Step 5: Run code by selecting "python app.py"
Step 6: Enjoy! :-)
